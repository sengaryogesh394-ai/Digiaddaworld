# **App Name**: DigiAddaWorld

## Core Features:

- Product Listings: Display a grid view of all digital products with details like image, title, and price, similar to Etsy and Sellfy.
- Product Filters: Allow users to filter products based on category and price.
- Product Details Page: Show detailed information about a selected product, including images, title, price, and description, with options to 'Add to Cart' and 'Buy Now'.
- Shopping Cart: A simple shopping cart to view selected items and proceed to checkout.
- User Authentication: Enable user login and signup using Firebase Authentication with email/password or Google.
- Protected Admin Routes: Restrict access to admin panel routes to only logged-in admin users.
- AI-powered product descriptions: Suggest and generate product descriptions with a tool that uses keywords related to the products.

## Style Guidelines:

- Primary color: Soft lavender (#E6E6FA) for a calming and elegant feel.
- Background color: Very light lavender (#F5F5FF) with a desaturated feel.
- Accent color: Muted violet (#B0B0D9) to provide contrast.
- Body font: 'PT Sans', a versatile humanist sans-serif suitable for body text, allowing easy readability.
- Headline font: 'Playfair', a modern serif to add elegance to headlines.
- Simple, minimalist icons to represent categories and actions.
- Responsive grid layout inspired by Etsy and Sellfy for product listings.
